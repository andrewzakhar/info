1 Загрузка и обработка данных¶

Исходные данные:¶

Дата-время, количества пассажиров JetRail, нового высокоскоростного железнодорожного сообщения от Unicorn Investors. Нам предоставляются данные за 2 года (август 2012 г. - сентябрь 2014 г.), и с помощью этих данных мы должны прогнозировать количество пассажиров на ближайшие 7 месяцев.

Исследуемое подмножество набора данных (август 2012 г. - декабрь 2013 г.)¶

Создание обучающего и тестового файла для моделирования. Первые 14 месяцев (август 2012 г. - октябрь 2013 г.) используются в качестве данных обучения, а следующие 2 месяца (ноябрь 2013 г. - декабрь 2013 г.) - данные тестирования.

4 Применение для выявления аномалий в тренировочной выборке¶

Задаем количество дней, по которым будем вычислять прогноз (длину окна): должно совпадать с количеством входов нейронной сети (input_dim первого слоя)¶

Создадим копию проверочной выборки $\hat{y}$, которую будем использовать для прогноза при помощи нейронных сетей¶

Алгоритм обучения и прогноза¶

  1. Выбираем первое окно длиной window_size, складываем его в стопку.
  2. Смещаем окно на единицу, складываем в стопку
  3. И так number_of_windows раз. Стопка из number_of_windows массивов длиной window_size используется для обучения нейронной сети и прогноза значения после последнего окна
  4. После этого вычисления повторяются со смещением на одно значение вперед.

Пример: window_size = 32, number_of_windows = 32

Шаг 1: начиная с первого значения окно длиной 32 значения смещается 32 раза и используется для прогноза 65 значения.

Шаг 2: начиная со второго значения окно длиной 32 значения смещается 32 раза и используется для прогноза 66 значения. и т.д.

Пример первой стопки¶

Обучение нейронной сети по предыдущей стопке, прогноз значения и вычисление аномалий¶
In [36]:
for i in range(0,len(train) - window_size - number_of_windows):
    for j in range(i, i + number_of_windows):
        if j ==i: 
            nn_input = train[j:j+window_size]['Count'].values
        else:         
            nn_input = np.vstack((nn_input,train[j:j+window_size]['Count'].values))
    nn_output = train['Count'][i+window_size:i+window_size+number_of_windows].values.reshape(-1,1) 
    model.fit(nn_input, nn_output, batch_size = 128, epochs = 10, verbose=0, validation_split = 0.1)
    new_base = train[i+number_of_windows:i+number_of_windows+window_size]['Count'].values.reshape(1,-1)
    y_hat['Count'][i+window_size+number_of_windows] = model.predict(new_base)[0]
    # print(abs(model.predict(new_base)[0]-train['Count'][i+number_of_windows+window_size]))
    if abs(model.predict(new_base)[0]-train['Count'][i+number_of_windows+window_size]) < 30:
        z_hat['Count'][i+number_of_windows+window_size]=np.NaN
